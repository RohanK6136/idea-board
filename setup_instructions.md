# Idea Board – Setup Instructions

## Prerequisites
- Node.js (v16+)
- npm

## Installation
1. Clone the repository: `git clone <your-repo-url>`
2. Install all dependencies: `npm run install`
3. Start the backend: `cd backend && npm start`
4. Start the frontend (dev): `cd frontend && npm run dev`
5. Open `http://localhost:5173`

## Production build
- Run `npm run build` from the root, then `npm start` to serve everything on port 3001.
